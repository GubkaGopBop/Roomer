# Roomer - Система веб-авторизации и личного кабинета для игры

## Описание проекта
Веб-платформа для управления игровым аккаунтом.
Позволяет пользователям регистрироваться, входить в систему, управлять профилем и просматривать игровую статистику.

### Модули:

#### 1.Модуль аутентификации
- `register.js` - Регистрация пользователей
- `login.js` - Авторизация
- `recovery.js` - Восстановление пароля

#### 2.Модуль личного кабинета
- `profile.js` - Игровая статистика

#### 3.Утилиты
- `generator.js` - Генератор игровых данных
